The Book Club

"The Book Club" is an online book club web application that I have designed and currently developing for my final year project. The web application aims to enable users to join an online book club by creating an account, vote for the book of the month, read and share their favourite quotes, join discussions and test their knowledge of books by completing quizzes.



The tools that I am using for the implementation of "The Book Club" include Visual Studio, ASP.Net Core, C-Sharp, HTML5, CSS and Entity Framework Core.
